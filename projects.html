Binary Search Tree with Value conversion card game simulation:
Implemented a binary search tree class by scratch in C++, which
stores int values. Input for the program is 2 .txt files holding
multiple playing card suits and values (strings). My program
parses through the .txt files, converts the string values into
int values, and inserts into their respective BSTs. Both BSTs are
then compared, and matching cards are removed from the BST. Removed
cards and remaining cards are translated back from int to string
and printed, with a description of if they were removed or not. SOURCE CODE
